# Gestion des tâches (reception-db)

Application Node.js / Express branchée sur ta base `reception-db` existante :

- Authentification basée sur un fichier `users.csv` (Nom;Prenom;email;password).
- Liste des chantiers depuis la table `chantiers`.
- Suivi des tâches via la table `interventions` + `floors` + `rooms`.
- Import Excel pour créer des interventions à partir d'un fichier de tâches.

Voir `README-render.md` pour la mise en place sur Render.
